# Text-Handler
<p>A program that performs various simple actions with text.</p><hr>
<p>Full list of supported actions:</p>
<li>Reversing</li>
<li>Translation from/to Pig Latin</li>
<li>Checking for being a palindrome</li>
<li>Counting different statistics</li>
<li>Ciphering/Deciphering (8 cipher types available)</li>
<li>Mathing regular expressions</li>
